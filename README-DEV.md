# Инструкция по запуску для разработки

## Проблема "failed to fetch"

Эта ошибка возникает, когда фронтенд не может подключиться к бэкенду.

## Решение

### 1. Создайте файл `.env.local` в корне проекта:

```bash
VITE_API_URL=http://localhost:3001
```

### 2. Запустите сервер (в отдельном терминале):

```bash
npm run server:dev
```

Или:

```bash
cd server
npm install  # если еще не установлены зависимости
npm run dev
```

### 3. Запустите фронтенд (в другом терминале):

```bash
npm run dev
```

## Проверка

1. Сервер должен быть доступен на `http://localhost:3001`
2. Фронтенд должен быть доступен на `http://localhost:5173`
3. В консоли браузера должно быть: `[API] API_URL: http://localhost:3001`

## Для продакшена

В продакшене `VITE_API_URL` не нужен (или должен быть пустым), так как фронтенд и бэкенд работают на одном домене.
